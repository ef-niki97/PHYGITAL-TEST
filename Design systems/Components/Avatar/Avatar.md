# Avatar

Часть [Design systems](../..). Инвентарь собран из Figma-файла [Components](https://www.figma.com/design/j5KLNIldIndVDcg5bsHZhC/Components), страница **Avatar**.

> Используется для визуального представления пользователя, сущности или источника контента.

## Размеры (px)

Все размеры — квадратные (высота = ширина), фиксированы для каждого шага шкалы:

| Size | Высота/Ширина |
|---|---|
| XS | 24px |
| S | 32px |
| M | 40px |
| L | 52px |
| XL | 60px |
| 2XL | 75px |
| 3XL | 100px |

## Семейства

### Avatar
- **Size**: 24 / 32 / 40 / 52 / 60 / 75 / 100
- **Type**: Image / Text / Icon
- **Circle**: On / Off

Полная матрица Size × Type × Circle задокументирована для всех 7 размеров.
