# Brand Design References

A curated collection of `DESIGN.md` files — plain-text design system docs extracted from real products — one per brand, under `design-md/<brand>/DESIGN.md`.

## How to use

1. Check `design-md/` for a folder matching the requested brand (folder names are lowercase, e.g. `airbnb`, `bmw-m`, `linear.app`, `x.ai`).
2. Read that brand's `DESIGN.md` for its documented tokens, typography, spacing, color palette, and component patterns.
3. Apply those concrete rules when generating or reviewing UI that should match the brand's visual language, rather than inventing generic defaults.
4. If no exact folder matches, look for the closest adjacent brand (e.g. same industry or product category) and say explicitly that it's an approximation.

## Available brands

Run `ls design-md` to list all 74 available brands.
